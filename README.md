# Information Visualization Assignment 3

## 3D Loan Risk Analysis Visualization

Live demo: https://idfepam.github.io/information-visualization-assignment-3/

Repository: https://github.com/idfepam/information-visualization-assignment-3.git

## About

TLoan approval data in a 3D scatter plot using Annual Income (X-axis), Credit Score (Y-axis), and Debt-to-Income Ratio (Z-axis).

I used not only D3 because D3 doesn't have 3D built-in, so I combined it with Three.js for the 3D visualization capabilities.

## Controls

- **Left click + drag**: Rotate
- **Right click + drag**: Pan
- **Scroll**: Zoom
- **Hover**: View details
